### 프로젝트 실행 방법   
#### Frontend
1. Frontend 파일 저장 위치로 이동
2. 아래 명령어 입력
```
npm update
npm start
```
#### Backend
1. Backend 파일 저장 위치로 이동
2. 아래 명령어 입력
```
gradlew bootrun
```
