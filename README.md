### 프로젝트 실행 방법   
1. CMD 열기(frontend, backend 2개 필요) - 1번 cmd, 2번 cmd
2. 1번 cmd - Frontend 파일 저장 위치로 이동
3. 아래의 커맨드 입력  
```
npm update
npm start
```
4. 2번 cmd - Backend 파일 저장 위치로 이동
5. 아래의 커맨드 입력  
```
gradlew bootrun
```
